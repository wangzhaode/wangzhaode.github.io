三月初了，北京却下了一场大雪。


![雪景](/assets/images/2026-03-04/snow.jpg)

雪断断续续下了一整天，不是那种暴风雪，就是一直飘着。傍晚去推电动车的时候，发现车座和把手上的雪已经冻成了一层薄冰，用手一摸，冷得刺骨。这都三月了，没想到还能下这么大。

早上刷到热搜，说林俊旸离职了。挺突然的，赶紧联系 Qwen 团队的同学聊了一下。电话那头，能感觉到他们挺伤感的。晚上又看了些新闻，感觉俊旸这次离开是不可挽回了，公司沟通上可能存在一定瑕疵。

下午两点多和 ARM 的同学在 B 区访客中心的星巴克当面聊了一下，了解了今年秋季芯片的新特性、SME2 技术分享等。端侧GPU AI算力这块值得期待。他买好了咖啡，我尝了一口——星巴克的玫瑰味，绿色的，超难喝，我一般不主动喝咖啡，这杯最后也没喝完。

![电动车上的雪](/assets/images/2026-03-04/snow-ebike.jpg)

工作方面，今天围绕 MNN 做了一些事情。尝试创建了几条 skills，让 Claude Code 能更顺畅地参与优化工作——算是给 AI 助手装了一套更趁手的工具。顺手把 OpenClaw 的底层模型从阿里云换成了联通的 GLM-5 Coding Plan，跑起来还不错。又调整了一些配置，整体交互更顺手了。另外给 MNN 加了个 AI PR Review 的功能，以后代码审查能省不少精力。

最让人开心的是，公司里那个困扰很久的 Claude API 缓存问题终于解决了。之前用 ccr code 做转换的时候，缓存命中率低得离谱，一句话就要烧掉四五块钱，用起来心惊肉跳，每次调用都得掂量再三。现在调整之后，成本降到了几毛钱一次——终于可以放开手脚用 Claude Opus 4.6 了。这种从"心疼"到"畅快"的转变，大概就是程序员特有的小确幸吧。

窗外雪还在下，屋里暖气正好。又是充实的一天。
